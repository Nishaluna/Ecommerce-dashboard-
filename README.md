E-Commerce Sales Dashboard

An interactive E-Commerce Sales Dashboard built to analyze and visualize business performance through key sales metrics and dynamic insights.

This dashboard provides a complete overview of sales activity, helping users track revenue, profit, order quantity, customer behavior, payment preferences, and regional performance.
